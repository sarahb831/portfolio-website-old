A selection of my projects

React Webstore with PayPal

A React-based webstore to rent out my goats as "lawnmowers". It uses the Context API for a single source of truth as well as React-Router and a PayPal sandbox.

React Webstore Link
React Webstore GitHub Repository

React Native Mobile Chat App

A chat application built using React Native. It uses the Expo.io open source platform and stores conversations in the Google Firebase Firestore Database. Users are authenticated with Google Firebase authentication. They can select pictures from the phone's image library, take and send pictures using the device's camera, and send their location in a map image. User conversations are also stored locally and can be viewed while the device is offline.

React Native Chat App GitHub Repository

Meetup App

A serverless, progressive web application (PWA) which uses the Meetup API to fetch upcoming events for a selected city. It is a React application which uses Amazon Web Services (AWS) Lambda and API Gateway for serverless functions for an authorization server with OAuth2 authentication flow. It employs a service worker which allows it to work offline. It also uses the Recharts library for data visualization with a scatter graph and pie chart.

Meetup App Link
Meetup App GitHub Repository

Angular Quiz App

A single-page application built using the Angular framework. It allows a user to choose one of several types of quizzes. After the user selects answers for each question, the results are displayed. The app was built using interpolation, data binding, services, routing, and Material Design.

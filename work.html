React Native Mobile Chat App

A chat application built using React Native, a framework for building iOS and Android apps using a single codebase. It also uses the Expo.io open source platform and stored conversations in the Google Firebase Firestore Database. Users are authenticated with Google Firebase authentication. They can select pictures from the phone's image library, take and send pictures using the device's camera, and send their location in a map image. User conversations are also stored locally and can be viewed while the device is offline.

MyMovies Full-Stack App

A fully functioning application with responsive single-page application website on the client-side and RESTful API on the server-side. It allows a user to register, login, view movies and create a list of favorite movies. The website was built using React, React-router, React-Bootstrap, and React Redux. The RESTful API was developed using Node.js, Express, MongoDB and Mongoose.

Meetup App

A serverless, progressive web application (PWA) which uses the Meetup API to fetch upcoming events for a selected city. It is a React application which uses Amazon Web Services (AWS) Lambda and API Gateway for serverless functions for an authorization server with OAuth2 authentication flow. It employs a service worker which allows it to work offline. It also uses the Recharts library for data visualization with a scatter graph and pie chart.
